# Somos Barrio — Claude Code Config

## Permisos
Este proyecto tiene permisos completos. Podés ejecutar cualquier comando bash,
leer y escribir cualquier archivo del directorio sin pedir confirmación.

## Comandos frecuentes
- `npm run dev` — levantar servidor
- `npx prisma migrate dev` — migrar DB
- `npx prisma generate` — regenerar cliente
- `npx prisma studio` — ver DB visualmente
- `npx tsc --noEmit` — chequear tipos
